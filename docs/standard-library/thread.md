# <a name="ltthreadgt"></a>&lt;thread&gt;

標準ヘッダーを含める\<スレッド > クラスを定義する**スレッド**とさまざまなサポート関数。

## <a name="syntax"></a>構文

```cpp
#include <thread>
```

## <a name="remarks"></a>Remarks

> [!NOTE]
> 使用してコンパイルされたコードで **/clr**、このヘッダーはブロックされます。

`__STDCPP_THREADS__`スレッドはこのヘッダーでサポートされていることを示すゼロ以外の値としてマクロが定義されます。
