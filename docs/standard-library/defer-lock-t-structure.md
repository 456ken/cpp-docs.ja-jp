# <a name="deferlockt-structure"></a>defer_lock_t 構造体

## <a name="syntax"></a>構文

```cpp
struct defer_lock_t;
```

## <a name="requirements"></a>必要条件

**ヘッダー:** \<mutex >

**名前空間:** std
