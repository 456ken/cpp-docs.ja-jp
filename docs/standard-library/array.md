# <a name="ltarraygt"></a>&lt;array&gt;

コンテナーのテンプレート クラスを定義します。**配列**といくつかのサポート用テンプレート。

## <a name="requirements"></a>必要条件

**Header:** \<array>

**名前空間:** std

> [!NOTE]
> \<配列 > ライブラリで使用することも、`#include <initializer_list>`ステートメント。
