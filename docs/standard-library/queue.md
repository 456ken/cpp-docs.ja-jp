# <a name="ltqueuegt"></a>&lt;queue&gt;

テンプレート クラスの priority_queue と queue、および複数のサポート テンプレートを定義します。

## <a name="requirements"></a>必要条件

**ヘッダー:** \<queue>

**名前空間:** std

> [!NOTE]
> \<キュー > ライブラリで使用することも、`#include <initializer_list>`ステートメント。
