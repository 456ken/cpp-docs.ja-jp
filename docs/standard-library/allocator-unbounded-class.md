# <a name="allocatorunbounded-class"></a>allocator_unbounded クラス

```cpp
template <class Type>
class allocator_unbounded;
```

### <a name="parameters"></a>パラメーター

|パラメーター|説明|
|---------------|-----------------|
|*Type*|アロケーターによって割り当てられた要素の型。|
