# <a name="ltcctypegt"></a>&lt;cctype&gt;

標準 C ライブラリ ヘッダー \<ctype.h> をインクルードし、関連する名前を `std` 名前空間に追加します。

## <a name="requirements"></a>必要条件

**ヘッダー:** \<cctype >

**名前空間:** std

## <a name="remarks"></a>Remarks

このヘッダーをインクルードすると、標準 C ライブラリ ヘッダーの外部リンケージで宣言された名前が、`std` 名前空間でも宣言されます。

## <a name="functions"></a>関数

```cpp
int isalnum(int c);
int isalpha(int c);
int isblank(int c);
int iscntrl(int c);
int isdigit(int c);
int isgraph(int c);
int islower(int c);
int isprint(int c);
int ispunct(int c);
int isspace(int c);
int isupper(int c);
int isxdigit(int c);
int tolower(int c);
int toupper(int c);
```
