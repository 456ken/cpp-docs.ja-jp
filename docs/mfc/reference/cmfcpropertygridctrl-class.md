# <a name="cmfcpropertygridctrl-class"></a>CMFCPropertyGridCtrl クラス

詳細についてにあるソース コードを参照してください、 **VC\\atlmfc\\src\\mfc** Visual Studio のインストールのフォルダー。

## <a name="syntax"></a>構文

```
class CMFCPropertyGridCtrl : public CWnd
```
