# <a name="ml-nonfatal-error-a2206"></a>ML の致命的でないエラー A2206

**式で演算子がありません。**

演算子がないために、式を評価できません。 このエラー メッセージは、前のプログラム エラーの副作用もあります。

このエラーは、次の行が生成されます。

```asm
value1 = ( 1 + 2 ) 3
```
