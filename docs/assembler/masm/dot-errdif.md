# <a name="errdifi"></a>.ERRDIF[[I]]

テキスト項目が異なる場合は、エラーを生成します。

## <a name="syntax"></a>構文

> .ERRDIF[[I]] *textitem1*, *textitem2* [[, *message*]]

## <a name="remarks"></a>Remarks

場合**は**は、比較では大文字小文字を区別します。
