# <a name="ml-nonfatal-error-a2039"></a>ML の致命的でないエラー A2039

**行が長すぎます**

ソース ファイルの行には、512 文字の制限を超えています。

行連結文字 (\) で複数の物理行が連結された場合、その結果、論理行は 512 文字に制限されます。
