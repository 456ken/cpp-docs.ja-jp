# <a name="ml-nonfatal-error-a2107"></a>ML の致命的でないエラー A2107

**暗黙の相手側ジャンプまたはラベルの近くへの呼び出しを含めることはできません。**

暗黙まで移動または別のセグメントでほぼラベルへの呼び出しが試行されました。
