# <a name="record-masm"></a>RECORD (MASM)

指定したフィールドで構成されるレコードの種類を宣言します。 *fieldname* 、フィールドの名前*幅*ビット単位の数を指定および*式*その初期値を提供します。

## <a name="syntax"></a>構文

> *recordname* RECORD *fieldname*:*width* [[= *expression*]] [[, *fieldname*:*width* [[= *expression*]] ]] ...
