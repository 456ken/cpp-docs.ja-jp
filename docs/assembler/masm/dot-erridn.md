# <a name="erridn"></a>.ERRIDN]

テキスト項目が同一の場合は、エラーを生成します。

## <a name="syntax"></a>構文

> .ERRIDN[[I]] *textitem1*, *textitem2* [[, *message*]]

## <a name="remarks"></a>Remarks

場合**は**は、比較では大文字小文字を区別します。
