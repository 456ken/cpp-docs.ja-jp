# <a name="requestedit"></a>requestedit

`OnRequestEdit` 通知がプロパティでサポートされることを示します。

## <a name="syntax"></a>構文

```cpp
[requestedit]
```

## <a name="remarks"></a>Remarks

**Requestedit** C++ 属性と同じ機能を持つ、 [requestedit](/windows/desktop/Midl/requestedit) MIDL 属性。
