# <a name="how-to-create-resources-c"></a>方法: リソース (C++) を作成します。

プロジェクト用のリソースを作成できます。

- リソース スクリプト ファイルを使用します。 リソースを追加する前に、この手順は必要があります。

- プロジェクトにリソースを追加します。 この手順は、使用する方法を示します、**リソース ビュー**します。

- リソース テンプレートを使用して、カスタマイズされたリソースを作成します。

> [!NOTE]
> **リソース エディター**と**リソース ビュー** Express エディションでは使用できません。

## <a name="use-resource-script-files"></a>リソース スクリプト ファイルを使用します。

作成して、新しいリソースをプロジェクトに追加する前にまず、リソース スクリプト (.rc) ファイルを作成する必要があります。

> [!NOTE]
> リソース スクリプトは、Visual Studio IDE に読み込まれる既存のプロジェクトにのみ追加できます。 リソース テンプレート (.rct ファイル) はいつでも作成できますが、プロジェクトの外側のスタンドアロン リソース スクリプトを作成することはできません。

### <a name="to-create-a-resource-script-file"></a>リソース スクリプト ファイルを作成するには

1. 選択、 **Visual C**フォルダー選択**リソース ファイル (.rc)** 右側のペインでします。

1. リソース スクリプト ファイルの名前を付けます、**名前**テキスト ボックスを選択します**オープン**します。

## <a name="use-resource-templates"></a>リソース テンプレートを使用します。

リソース テンプレートは、.rct ファイルとして保存したカスタマイズされたリソースです。 リソース テンプレートは、リソースを作成するための開始点として機能します。 リソース テンプレートは、その他のリソースまたはリソースを標準のコントロールなどの機能を共有したり、要素の繰り返しのグループの開発に時間を節約します。 など複数のダイアログ ボックスで、会社のロゴ アイコンで、[ヘルプ] ボタンを追加するには、新しいダイアログ ボックスのテンプレートを作成しで、[ヘルプ] ボタンおよびロゴでカスタマイズできます。

リソース テンプレートをカスタマイズするには後で変更を保存、テンプレート フォルダーまたはインクルード パスでは、指定された場所でリソースの種類の下に、新しいリソース テンプレートが表示されるよう、**リソースの追加** ダイアログ ボックス。 必要に応じて何度でも、新しいリソース テンプレートを使用することができますようになりました。

> [!NOTE]
> リソース エディターによって一意のリソース ID が自動的に指定されます。 変更することができます、[リソース プロパティ](../windows/changing-the-properties-of-a-resource.md)に応じて。

> [!NOTE]
> 言語固有のテンプレート ファイルをメイン テンプレートのディレクトリのサブディレクトリに配置します。 たとえば、英語版のみのテンプレート ファイル移動.\\< リソース テンプレート ディレクトリ\>\1033 します。 Visual Studio 検索 \Program Files\Microsoft Visual Studio で新しい .rct ファイル\<バージョン\>\VC\VCResourceTemplates、Visual Studio \Program Files\Microsoft\<バージョン > \VC\VCResourceTemplates\\< LCID\> (など、1033 は英語の LCID)、または任意の場所に、[インクルード パス](../windows/how-to-specify-include-directories-for-resources.md)します。 別の場所の .rct ファイルを格納する場合は、インクルード パスに場所を追加する必要があります。
