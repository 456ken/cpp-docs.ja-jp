# <a name="mbrtoc16-mbrtoc32"></a>mbrtoc16、mbrtoc32

半角文字列の最初のマルチバイト文字を等価の UTF-16 や UTF-32 の文字に変換します。

## <a name="syntax"></a>構文

```C
size_t mbrtoc16(
   char16_t* destination,
   const char* source,
   size_t max_bytes,
   mbstate_t* state
);

size_t mbrtoc32(
   char32_t* destination,
   const char* source,
   size_t max_bytes,
   mbstate_t* state
);
```
