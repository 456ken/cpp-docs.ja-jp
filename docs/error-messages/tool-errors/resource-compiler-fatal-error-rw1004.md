# <a name="resource-compiler-fatal-error-rw1004"></a>リソース コンパイラの致命的なエラー RW1004

予期しないファイルの終わり

このエラーは、ライン フィード文字とテキスト ファイルの最後の行で文字をキャリッジ リターンがないために発生することができます。
