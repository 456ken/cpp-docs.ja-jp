# <a name="resource-compiler-warning-rc4214"></a>リソース コンパイラの警告 RC4214

コード ページが無効です無視されます。

.Rc ファイルには、codepage 引数が含まれているし、指定されたコードページは無効です。 参照してください[IsValidCodePage](/windows/desktop/api/winnls/nf-winnls-isvalidcodepage)詳細についてはします。
