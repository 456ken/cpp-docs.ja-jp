# <a name="compiler-error-c2571"></a>コンパイラ エラー C2571

'function': 仮想関数は、共用体 'union' にすることはできません

仮想関数を持つ共用体が宣言されています。 クラスまたは構造体でのみ仮想関数を宣言することができます。  考えられる解決策:

1. クラスまたは構造体、共用体に変更します。

1. 非仮想関数を作成します。

次の例では、C2571 が生成されます。

```
// C2571.cpp
// compile with: /c
union A {
   virtual void func1();   // C2571
   void func2();   // OK
};
```
