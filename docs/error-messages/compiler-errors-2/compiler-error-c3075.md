# <a name="compiler-error-c3075"></a>コンパイラ エラー C3075

'instance': 参照型 'type' のインスタンスを値型に埋め込むことはできません

値型に参照型のインスタンスを含めることはできません。

詳細については、[参照型の C++ スタック セマンティクス](../../dotnet/cpp-stack-semantics-for-reference-types.md)を参照してください。

## <a name="example"></a>例

次の例では C3075 が生成されます。

```
// C3075.cpp
// compile with: /clr /c
ref struct U {};
value struct X {
   U y;   // C3075
};

ref struct Y {
   U y;   // OK
};
```
