# <a name="compiler-error-c3228"></a>コンパイラ エラー C3228

'function' : 'param' のジェネリック型引数を 'type' にすることはできません。値型またはハンドル型にする必要があります

正しくない型がジェネリック型引数として渡されました。

次の例では C3228 が生成されます。

```
// C3228.cpp
// compile with: /clr
class A {};

value class B {};

generic <class T>
void Test() {}

ref class C {
public:
   generic <class T>
   static void f() {}
};

int main() {
   C::f<A>();   // C3228
   C::f<B>();   // OK

   Test<C>();   // C3228
   Test<C ^>();   // OK
}
```
