# <a name="compiler-error-c2792"></a>コンパイラ エラー C2792

'super': このキーワードの後に ':: '

キーワードに続くことができますのみトークン`__super`は`::`します。

次の例では、C2792 が生成されます。

```
// C2792.cpp
struct B {
   void mf();
};

struct D : B {
   void mf() {
      __super.();   // C2792

      // try the following line instead
      // __super::mf();
   }
};
```
