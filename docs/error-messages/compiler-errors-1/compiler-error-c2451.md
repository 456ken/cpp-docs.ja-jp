# <a name="compiler-error-c2451"></a>コンパイラ エラー C2451

条件式の型 'type' は無効です。

条件式は、整数型に評価されます。

次の例では、C2451 が生成されます。

```
// C2451.cpp
class B {};

int main () {
   B b1;
   int i = 0;
   if (b1)   // C2451
   // try the following line instead
   // if (i)
      ;
}
```
