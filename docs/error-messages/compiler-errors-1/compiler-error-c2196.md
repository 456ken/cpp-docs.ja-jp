# <a name="compiler-error-c2196"></a>コンパイラ エラー C2196

case の値 'value' は既に使用されています。

switch ステートメントで、同じ case 値を複数回使用します。

次の例では C2196 が生成されます。

```
// C2196.cpp
int main() {
   int i = 0;
   switch( i ) {
   case 0:
      break;
   case 0:   // C2196
   // try the following line instead
   // case 1:
      break;
   }
}
```
