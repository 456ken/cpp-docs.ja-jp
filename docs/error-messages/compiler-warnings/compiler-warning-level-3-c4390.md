# <a name="compiler-warning-level-3-c4390"></a>コンパイラの警告 (レベル 3) C4390

';'。 空の被制御文が見つかりました。目的ですか。

手順が含まれていない制御ステートメントの後にセミコロンが見つかりました。

マクロにより C4390 を取得する場合は使用、[警告](../../preprocessor/warning.md)プラグマ マクロを含むモジュールで C4390 を無効にします。

次の例では、C4390 が生成されます。

```
// C4390.cpp
// compile with: /W3
int main() {
   int i = 0;
   if (i);   // C4390
      i++;
}
```
